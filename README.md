# Тестовое задане фирмы Webest

**Версия Node.js**: v24.1.0

## Установка зависимостей

```sh
npm install
```

### Компиляция и сборка проекта в режиме hot-reload

```sh
npm run dev
```

### Сборка проекта для продакшена

```sh
npm run build
```

### Форматирование

```sh
npm run format
```

### Линтинг

```sh
npm run lint
```
